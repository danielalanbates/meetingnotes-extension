# MeetingNotes

A powerful Chrome extension for taking meeting notes with templates, themes, export, and sharing features.

**Version**: 1.0.3  
**License**: Polyform Noncommercial 1.0.0

## Features

✅ **Clean Interface**
- Simple, distraction-free note-taking
- Basic text formatting (bold, italic, underline)
- Bullet and numbered lists
- Template support (General, Standup, 1-on-1, Retrospective)

✅ **Meeting Detection**
- Auto-detects meetings on Zoom, Google Meet, Teams, and Webex
- Shows meeting status in the sidebar

✅ **Export Options**
- Export as Plain Text (.txt)
- Export as Markdown (.md)
- Export as HTML (.html)

✅ **History**
- Automatic save to history
- Browse and reload past notes

✅ **Lightweight**
- No external dependencies
- No AI API costs
- No premium tiers
- Completely free

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the `Simple_MeetingNotes` folder

## Usage

1. Join a meeting on Zoom, Google Meet, Teams, or Webex
2. Click the extension icon or open the side panel
3. Start taking notes
4. Click "Save Note" to save to history
5. Click "Export" to download your notes

## Templates

- **Blank** - Start fresh
- **General Meeting** - Agenda, discussion, action items
- **Daily Standup** - Yesterday, today, blockers
- **One-on-One** - Check-in, goals, feedback
- **Retrospective** - What went well, what to improve, action items

## Privacy

- All notes stored locally in your browser
- No data sent to external servers
- No tracking or analytics
- Open source and transparent
